2 : Appling css and js file

  • After extracting (zip file)
  • you can see directory structure like

  •  

    if you wan't add any extra css or js file than put in to css folder if file type is *.css or js folder if file type is *.js

    put link in to html file like this

    custom.css is your css file name & custom.js is your js name


    Source File (CSS)

    <!-- css  -->
=================

<!-- Font awesome CSS -->
<link rel="stylesheet" type="text/css" href="css/font-awesome.min.css" />

<!-- Bootstrap CSS -->
<link rel="stylesheet" type="text/css" href="css/bootstrap.css" />

<!-- Magnific-popup CSS -->
<link rel="stylesheet" type="text/css" href="css/magnific-popup.css" />

<!-- Jquery-ui CSS -->
<link rel="stylesheet" type="text/css" href="css/jquery-ui.css" />

<!-- Owl Carousel CSS -->
<link rel="stylesheet" type="text/css" href="css/owl.carousel.css" />

<!-- Fotorama CSS -->
<link rel="stylesheet" type="text/css" href="css/fotorama.css" />

<!-- Custom CSS -->
<link rel="stylesheet" type="text/css" href="css/custom.css" />

<!-- Responsive CSS -->
<link rel="stylesheet" type="text/css" href="css/responsive.css" />

    <!-- Jquery  -->
